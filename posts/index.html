如何使用GitHub

如何使用GitHub.pages预览页面 登入Github,新建repo仓库 在左上角找到新建仓库,点击创建. 自定义仓库名称（无需做任何选项的操作）,确定仓库的创建. 进入新建的repo仓库,在右边tab中找到settings 点击进入settings选项,下滑找到GitHub Pages栏进行设置选择. 在source选项一栏中下拉选择master branch,然后点击save. 你会在GitHub栏看到一个自动生成的预览页面网址,就成功了. 想要预览html页面 将你需要预览展示的html页面文件加入新建的repo中. 在生成的预览网址后面加上想访问的html路径名称即预览. （例：我的html路径名称为git-demo-1/index.html,则可以访问http://zheng-en.xyz/git-demo-1/index.html.) 大公告成！😁😁😁 谢谢观看！

如何用hugo搭建个人博客

Hugo制作博客步骤 下载Hugo (windows) windows安装地址 解压并运行 将解压后的hugo.exe文件放入某盘下的software\hugo\hugo.exe中 将hugo.exe的文件路径添加至（我的电脑>属性>高级系统设置>环境变量）中的path选项中 打开cmder，运行hugo server ，确认是否正常运行。 选择一个空目录建立网站目录 登录Hugo,点击quick start，跟随网页的引导步骤建立站点。 最终运行hugo之后，得到一个public的目录生成，标志你的博客站点已经建立成功。 将个人博客推到网上 在你的网站目录里生成一个.gitignore的文件，将public目录写进里面。 在进入public目录进行初始化仓库，将网页blog内容上传至github，生成博客站点网址。 cd public git init git add git commit -v git remote add origin (github的仓库地址) git push -u origin master 之后想更新博客内容，可以在重新新建一个md文件进行编写。（要记得每次更新完内容之后记得hugo） hugo new posts/我的第二篇博客.md hugo ga gc git push 这样就完成了利用hugo生成博客，我还可以随时更新博客内容。是不是很方便呢，可以记录我们的笔记、想法、和有趣的事情，赶紧行动起来去建立一个属于你个人的博客！ c😁😁😁 Thanks For Watching ！ (可标题点击进入查看)🐶
